## Implemented

* Statements:
    * global/local variables
    * global/local functions
    * function call
    * break
    * while - do
    * if - then - else - if - end
    * for - do - end
* Expressions:
    * nil, true, false
    * number
    * string
    * function expression (lambda)
    * tableconstruct
    * power operator
    * unary operators
    * mul/div operators
    * add/sub operators
    * comparison operators
    * and/or operators
